# Sales-Prediction-Model
This project focuses on building a machine learning model to predict product sales based on historical data.

The goal is to analyze sales patterns and develop predictive models that can help forecast future sales performance.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, and regression model training for sales prediction.

Tech stack: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

Key steps:

Data cleaning and preprocessing
Exploratory data analysis (EDA)
Feature engineering for sales prediction
Regression model training
Model evaluation and performance metrics (MAE, RMSE, R²
